Short description

El nick de usuario de github sale de luck + 59 41 que en ASCII es luck;)

Me gusta habitar en Github. Aquí me siento como en casa. Github es una plataforma para desarrolladores donde pueden subir el código que realizan en proyectos de tal forma que cualquiera pueda verlo. Es decir, se fomenta la creación de código open source. Además, Github ofrece un dominio gratuito para alojar proyectos que sean html, css y javascript, de forma que se pueda ver el resultado sin tener que descargarlo.

Nunca me gustaron los portfolios tradiciones dado su carácter de ofrecerte como un producto más dentro de él. Por ese motivo, y dada la necesidad de tener uno, cree un espacio en el que se pudiesen ver los diferentes trabajos agrupados en carpetas. Aquí empezó el verdadero ejercicio. Desde pequeño me fascinó el mundo open source y en especial Linux y aquí encontré una oportunidad de diseñar una interface para un sistema operativo en base Linux con la premisa de, ya que uso internet, voy a tomar como referencia los nodos con los que se representa internet Por eso, todos mis trabajos se agrupan en carpetas conectadas como si se tratasen de nodos, atribuyéndoles un peso en función de su contenido.

La primera vez que conocí los nodos tenía apenas diez años. Ahora no podría imaginar mi futuro laboral sin ellos. Ellos me han ayudado a simular este sistema operativo en web, planteando las mismas posibilidades que en una web al uso. Gracias a ellos, en un futuro podré compartir contenido de forma global o seleccionando a las personas con las que se quiere compartir, generar chats, etc..., ya que cada persona nos conectamos a un nodo de la web cuando accedemos a internet. Esta motivación a generar una red social viene de la teoría de los seis grados de separación, cuya representación más habitual son nodos. Mi pequeña luca por generar un espacio distinto dentro de la red permite que el usuario se detenga, tomándose el tiempo que necesite para visualizar todo el contenido, ofreciendo una forma de interactuar más calmada y pausada que plataformas con un diseño más convencional, sin perder el dinamismo ya que siempre van a haber elementos en movimiento.
